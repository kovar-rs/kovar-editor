# Kovar Editor

A visual UI designer for the [Kovar](https://github.com/KercyDing/kovar) framework, built with [tldraw](https://tldraw.dev).

## Features

- Visual drag-and-drop UI design
- Export to Kovar Schema / HTML
- Component nesting support
- Real-time code preview

## Getting Started

```bash
pnpm install
pnpm dev
```

## Acknowledgments

This project uses [tldraw](https://tldraw.dev) as its canvas engine. tldraw is an excellent open-source infinite canvas library.
